# Maná Machado
> E ai, Maná!?

## Dependencies

* node 14+
* next 10+

### Install

```sh
npm install
```

### Test
```sh
npm test
```

### Run

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Start

Do build before start project

```sh
npm start
```

## Deploy

Is easy! When your PR is merged, a github action publishes the new version on vercel.com :sunglasses:
